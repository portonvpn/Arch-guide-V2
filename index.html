Arch guide v3 — now with actual good DE

for hp mfs and goonersss · ml4w hyprland edition
Before booting
Step 1
Disable Secure Boot
Spam F10 on startup → Security → Secure Boot → Disabled → F10 save. Plug in USB, spam F9 → pick USB → pick Arch ISO.
You're in when you see root@archiso ~ #
Live USB
Step 2
Connect WiFi
iwctl
station wlan0 scan
station wlan0 get-networks
station wlan0 connect "YourWiFiName"
exit
ping -c 3 archlinux.org
Password cursor won't move — normal. Ctrl+C to stop ping.
Step 3
Find your disk
lsblk
Find your drive by size — probably sda. The USB shows up too, ignore it. Guide uses /dev/sda everywhere.
Step 4
Wipe everything first
This deletes everything on the disk. No going back.
wipefs -af /dev/sda
sgdisk -Z /dev/sda
Step 5
Partition
cfdisk /dev/sda
Arrow keys to move, Enter to select:

1. Select gpt if asked
2. New → 512M (boot)
3. Free space → New → 4G (swap)
4. Free space → New → Enter (rest, your actual system)
5. Go to 512M → Type → EFI System
6. Go to 4G → Type → Linux swap
7. Write → type yes → Quit
Step 6
Format
mkfs.fat -F32 /dev/sda1
mkswap /dev/sda2
mkfs.ext4 /dev/sda3
Step 7
Mount
mount /dev/sda3 /mnt
mount --mkdir /dev/sda1 /mnt/boot
swapon /dev/sda2
Install
Step 8
Pacstrap — main install
pacstrap -K /mnt base linux \
  linux-firmware intel-ucode \
  networkmanager sudo nano \
  git base-devel grub \
  efibootmgr mesa \
  vulkan-intel \
  intel-media-driver \
  pipewire pipewire-alsa \
  pipewire-pulse wireplumber \
  xdg-user-dirs
Takes 5–15 min depending on WiFi. Don't close terminal.
Step 9
fstab + chroot
genfstab -U /mnt >> /mnt/etc/fstab
arch-chroot /mnt
Prompt changes after chroot — you're now inside your new system.
Configure (inside chroot)
Step 10
Timezone
ln -sf /usr/share/zoneinfo/Asia/Kolkata \
  /etc/localtime
hwclock --systohc
Step 11
Language
nano /etc/locale.gen
Ctrl+W → search en_US.UTF-8 → find #en_US.UTF-8 UTF-8 → delete the # → Ctrl+X → Y → Enter
locale-gen
echo "LANG=en_US.UTF-8" > /etc/locale.conf
Step 12
Hostname
echo "myarch" > /etc/hostname
Replace myarch with any name, no spaces.
Step 13
Bootloader
grub-install \
  --target=x86_64-efi \
  --efi-directory=/boot \
  --bootloader-id=GRUB
grub-mkconfig -o /boot/grub/grub.cfg
Should say "Found linux image" — good.
Step 14
Root password + your user
passwd
useradd -mG wheel yourname
passwd yourname
EDITOR=nano visudo
In visudo: Ctrl+W → search %wheel ALL → find # %wheel ALL=(ALL:ALL) ALL → delete the # → Ctrl+X → Y → Enter
Step 15
Enable network + reboot
systemctl enable NetworkManager
exit
umount -R /mnt
reboot
Pull the USB out when screen goes dark!
First boot — install ML4W hyprland
Step 16
Connect WiFi
nmtui
Activate a connection → pick WiFi → enter password → OK → Esc
Step 17
Install paru (AUR helper)
Make sure you're logged in as your normal user, not root. If you see root@: su yourname then cd ~
cd /tmp
git clone https://aur.archlinux.org/paru.git
cd paru
makepkg -si
Press Enter when asked "Proceed with installation?"
Step 18
Install ly (login manager)
Tiny, fast, looks like a retro terminal login. No bloat.
paru -S ly
sudo systemctl enable ly
That's the entire login manager install. Two commands.
Step 19
Install ML4W Hyprland
paru -S ml4w-hyprland
Will pull in hyprland + a lot of deps. Takes a while. Not frozen.
ml4w-hyprland-setup
An interactive wizard starts — follow the prompts to pick your config. It handles everything: waybar, rofi, themes, the works.
Step 20
Reboot into your setup
sudo reboot
ly greets you on the terminal login screen. Log in → Hyprland loads. You're done.
After install
# Update everything
paru -Syu

# Set up home folders
xdg-user-dirs-update

# If WiFi drops after reboot
nmtui
